# Contributing to JobSpy-Streamlit

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/yourusername/JobSpy-Streamlit.git`
3. **Create** a feature branch: `git checkout -b feature/amazing-feature`
4. **Make** your changes
5. **Test** locally: `streamlit run app.py`
6. **Commit**: `git commit -m 'Add amazing feature'`
7. **Push**: `git push origin feature/amazing-feature`
8. **Create** a Pull Request

## Development Setup

```bash
# Clone repository
git clone https://github.com/yourusername/JobSpy-Streamlit.git
cd JobSpy-Streamlit

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

## Code Style

- Follow PEP 8 guidelines
- Use meaningful variable names
- Add docstrings to functions
- Keep functions focused and small

## Testing

- Test new features locally before submitting PR
- Run the full app from start to finish
- Test with different input values
- Check for error messages

## Reporting Issues

- **Check existing issues** before creating a new one
- **Describe the problem** clearly
- **Include steps to reproduce**
- **Attach logs** if applicable

## Areas for Contribution

- [ ] Bug fixes
- [ ] Performance improvements
- [ ] New job board integrations
- [ ] UI/UX improvements
- [ ] Documentation
- [ ] Tests

## Pull Request Guidelines

- Keep PRs focused and manageable
- Add a clear title and description
- Reference related issues
- Wait for review before merging
- Be open to feedback

## Questions?

Feel free to open an issue with the `question` label or start a discussion.

Happy contributing! 🎉
